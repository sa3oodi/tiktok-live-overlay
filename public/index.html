Setup Instructions:

  1. Enter a TikTok username above and click "Generate URL"
  2. Copy the generated URL
  3. In OBS Studio, add a "Browser Source"
  4. Paste the URL and set Width: 400px, Height: 500px
  5. Enable "Shutdown source when not visible" and "Refresh browser when scene becomes active"
